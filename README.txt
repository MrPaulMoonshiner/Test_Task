До вашої уваги тестове завдання "Гостьова книга" для компанії Dreamext.

Для розгортання проекту на власному сервері необхідно зробити наступні маніпуляції:

1. Завантажте файли зі зразком git clone https://github.com/MrPaulMoonshiner/Test_Task.git .

2. Виконати команду composer install

3. Налаштувати конфігурацію БД в файлі.

    3.1 Вибрати драйвер БД.

    3.2 Задати доступпи до БД

4. Почистить кеш.

Після цього проекту готовий для використання.
