До вашої уваги тестове завдання "Гостьова книга" для компанії Dreamext.

Для розгортання проекту на власному сервері необхідно зробити наступні маніпуляції:

1. Завантажте файли, напишіть в командній строці: git clone https://github.com/MrPaulMoonshiner/Test_Task.git

2. В корені проекту виконайте команду composer update

3. Налаштуйте конфігурацію БД в файлі ".evn"

    3.1 Вибрати драйвер БД. (mysql або sqlite)

    3.2 Задати доступпи до БД (Логін Пароль)

    3.3 Задати ім'я БД

4. Виконайте команду   php bin/console doctrine:schema:create   для створення БД

5. В новоствореній БД знайдіть таблицю User та створіть користувача.

    5.1 В полі role  задайте ролі корситувача

        5.1.1 При реєстрації за замовчуванням задається ROLE_USER
        5.1.2 Для адміністратора треба додати в це поле через кому ще дві ролі:
               ROLE_ADMIN
               ROLE_SUPER_ADMIN

6. Очистіть кеш.

7. Проект готовий до використання.
