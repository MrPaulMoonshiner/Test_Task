До вашої уваги тестове завдання "Гостьова книга" для компанії Dreamext.

Для розгортання проекту на власному сервері, потрібно зробити наступні маніпуляції:

1. Завантажити файли за допомогою git clone https://github.com/MrPaulMoonshiner/Test_Task.git .

2. Виконати команду composer install

3. Налаштувати конфігурацію БД в файлі .evn:

    3.1 Вибрати драйвер БД.

    3.2 Задати доступи до БД

4. Почистити кеш.

Після цього проект готовий для використання.
